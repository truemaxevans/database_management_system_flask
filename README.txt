Database Management System

Database Management System "DBMS" is a software that allows users to create and manage tables. It is a collection of interrelated data and provides a way to store, modify, and extract data from the database.

STACK: Flask, Python, SQLite3, Pandas, HTML, CSS, SQL

To run the project:

1. Clone the repository
2. Install the requirements
3. To run the app, run manage.py file (it will run three files: createdb.py, insert_data.py, get_data.py)
P.S if you would like to follow step by step, run createdb.py, insert_data.py, get_data.py files separately
